# 🚀 快速开始 - 针织引导

## 立即体验

1. **访问应用**
   - 在应用页面找到"🧶 针织引导"卡片
   - 点击"在线演示"或"访问项目"按钮
   - 或直接访问：`/_apps/knitting-guide/index.html`

2. **准备图片**
   - 选择一张简单的图片（建议：图标、logo、简单图形）
   - 推荐尺寸：500x500 到 800x800 像素
   - 格式：JPG、PNG均可

3. **基础流程**
   ```
   选择图片 → 设置参数 → 生成图案 → 开始引导 → 逐步编织
   ```

## ⚙️ 推荐设置

### 🎯 初学者（第一次使用）
- **图片**：简单的几何图形、emoji表情
- **网格大小**：30-40 像素
- **颜色数量**：6-8 种
- **预计时间**：10-15 分钟

### 🎨 进阶用户
- **图片**：风景照、人像
- **网格大小**：20-30 像素
- **颜色数量**：10-15 种
- **预计时间**：30-60 分钟

### 🏆 专家级
- **图片**：高清照片
- **网格大小**：10-20 像素
- **颜色数量**：15-20 种
- **预计时间**：2+ 小时

## 💡 最佳实践

### 选择合适的图片
✅ **推荐**：
- 对比度高的图片
- 主体明确的照片
- 简单背景
- 色彩鲜明

❌ **避免**：
- 过于复杂的背景
- 低对比度的图片
- 过多细节的照片

### 优化参数设置
- **开始简单**：先用大网格试验
- **逐步调整**：找到最佳平衡点
- **预览效果**：生成后检查图案
- **适时调整**：不满意可重新生成

### 编织技巧
1. **按顺序进行**：跟随高亮提示
2. **核对颜色**：确认当前颜色
3. **定期保存**：避免丢失进度
4. **休息放松**：长时间编织需要休息

## 🎮 快捷操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 完成当前方块 | `空格` 或 `Enter` | 标记完成并前进 |
| 上一步 | `←` | 回到上一个方块 |
| 下一步 | `→` | 前进到下一个方块 |
| 点击方块 | 鼠标点击 | 完成当前高亮方块 |

## 🔧 故障排除

### 图片无法加载？
- 检查文件格式（支持 JPG、PNG）
- 确认文件大小（建议 < 5MB）
- 尝试刷新页面

### 生成图案很慢？
- 减小网格大小
- 减少颜色数量
- 使用较小的图片

### 颜色不准确？
- 增加颜色数量
- 使用对比度更高的图片
- 预处理图片以增强颜色

### 进度丢失？
- 定期点击"保存进度"
- 不要清除浏览器缓存
- 同一浏览器访问
